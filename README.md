

A simple Flask web app that recommends movies based on genre selection.

 About
This project suggests movies to users based on their favorite genre. 
It uses Python, Flask and a CSV file with movie data.

Features
- Select a genre and get movie recommendations
- Shows Movie Name, Genre, and Rating
- Clean UI with movie posters
- Data stored in `movies.csv`

 Tech Stack
- **Backend**: Python, Flask, Pandas
- **Frontend**: HTML, CSS
- **Data**: CSV file
